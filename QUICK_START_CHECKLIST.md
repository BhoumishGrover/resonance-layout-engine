# ✅ Quick Start Checklist

Copy this checklist and check off items as you complete them!

## Phase 1: Cloudinary Setup
```
[ ] Go to https://cloudinary.com/users/register_free
[ ] Sign up with email & verify
[ ] Complete welcome questions
[ ] See the Cloudinary Dashboard
[ ] Find "Account Details" section
[ ] Copy Cloud Name (looks like: dxxxxxx)
[ ] Copy API Key (looks like: 123456789012345)
[ ] Copy API Secret (click eye icon to reveal)
```

## Phase 2: Project Configuration
```
[ ] Open terminal
[ ] Navigate to project: cd /Users/RDL/site-2/resonance-layout-engine
[ ] Run: cp .env.example .env
[ ] Open .env in editor
[ ] Paste Cloud Name into VITE_CLOUDINARY_CLOUD_NAME=
[ ] Paste API Key into VITE_CLOUDINARY_API_KEY=
[ ] Paste API Secret into VITE_CLOUDINARY_API_SECRET=
[ ] Save .env file
```

## Phase 3: Upload Assets
```
[ ] Go to Cloudinary → Media Library
[ ] Click "Create Folder"
[ ] Name it: resonance-digital
[ ] Open that folder
[ ] Create subfolders: logos, videos, work, brands, team
[ ] Download/access your Google Drive assets
[ ] Drag and drop files into appropriate folders
[ ] Wait for uploads to complete (watch progress bar)
[ ] All files show green checkmarks
```
